description: Respostas do agente devem ter gramática correta, sem abreviações de internet (vc, tbm, pq), mas expressões informais faladas (Show, tá, pra) são permitidas

Nunca usar abreviações de internet nas respostas do agente de vendas: "vc", "vcs", "pq", "tbm", "oq", "td", "blz", "cmg". Sempre escrever as palavras completas.

**Why:** O usuário quer que as respostas sejam gramaticalmente corretas, mesmo sendo informais. Abreviações passam impressão de desleixo.

**How to apply:** Ao editar prompts ou mensagens do agente, usar palavras completas (você, vocês, porque, também). Expressões coloquiais faladas como "Show", "tá", "pra", "né", "rs" continuam permitidas pois representam fala natural, não abreviação escrita.
